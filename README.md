Project Overview

The Loan Application System is a web-based application developed to simplify and automate the loan application process. It allows users to apply for loans, track their application status, and enables administrators to manage and approve/reject applications efficiently.

*Features

- User Registration & Login

- Apply for Loan
  
- Track Loan Status

- Secure Authentication

- Admin Dashboard

- Loan Approval / Rejection

- Database Management



🛠️ Tech Stack
Frontend: HTML.CSS,Bootstrap

Backend:Java (Advanced Java)

Frameworks: Spring Boot 
           Spring Data JPA

Database:MySQL

Tools: Eclipse IDE.GitHub
